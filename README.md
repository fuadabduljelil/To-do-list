# 📝 To-Do List

A simple and responsive To-Do List application for managing daily tasks. Users can add, edit, complete, and delete tasks. The app stores tasks locally so they remain available after refreshing the page.

## ✨ Features

* Add new tasks
* Edit tasks
* Mark tasks as completed
* Delete tasks
* Save tasks with Local Storage
* Responsive design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
